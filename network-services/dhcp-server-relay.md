# DHCP Server and Relay

## Objective
Automatically assign IP addresses to hosts
across VLANs using DHCP.

## Topology
- DHCP server
- Layer 3 switch with ip helper-address

## Key Configurations
- DHCP pools
- Relay configuration

## Verification
- show ip dhcp binding
- Client IP assignment

## Outcome
Hosts received valid IP configurations
successfully.
