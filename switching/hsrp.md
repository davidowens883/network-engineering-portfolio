# HSRP – Gateway Redundancy

## Objective
Provide a redundant default gateway using
Hot Standby Router Protocol (HSRP).

## Topology
- Two Layer 3 devices
- VLAN 10 – 192.168.10.0/24

## Key Configurations
- HSRP group setup
- Priority and preemption

## Verification
- show standby brief
- Failover testing

## Outcome
Gateway redundancy ensured uninterrupted
client connectivity.
